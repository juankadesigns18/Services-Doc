¿Qué Es Amazon Web Services (AWS)?

Amazon Web Services (AWS) es una plataforma integral de servicios de computación en la nube proporcionada por Amazon. AWS ofrece una amplia gama de servicios que permiten a las empresas y desarrolladores almacenar datos, procesar información, gestionar bases de datos, desplegar aplicaciones y mucho más, sin necesidad de invertir en infraestructura física.

Beneficios de usar la nube de AWS ⇛

Facilidad de uso: AWS permite a proveedores de aplicaciones, software y distribuidores hospedar aplicaciones de manera rápida y segura, ya sean nuevas o existentes basadas en SaaS. Puede acceder a la plataforma de hospedaje de AWS mediante la consola de administración o las API de servicio web bien documentadas.

Flexible: AWS le permite seleccionar el sistema operativo, el lenguaje de programación, la plataforma web, la base de datos y otros servicios necesarios. Con AWS, puede cargar el software y servicios que su aplicación requiere en un entorno virtual, facilitando la migración de aplicaciones existentes y la creación de nuevas soluciones.

Escalabilidad y alto desempeño: Con las herramientas de AWS, Auto Scaling y Elastic Load Balancing, su aplicación podrá ampliarse o reducirse según la demanda. Gracias al respaldo de la sólida infraestructura de Amazon, tendrá acceso a los recursos informáticos y de almacenamiento siempre que los necesite.

Mapa de la Infraestructura global de AWS

La nube de AWS abarca 105 zonas de disponibilidad en 33 regiones geográficas, con el anuncio de que existen planes para crear 21 zonas de disponibilidad más y 7 regiones de AWS más en Malasia, México, Nueva Zelanda, el Reino de Arabia Saudí, Tailandia, Taiwán y la nube soberana europea de AWS.

La Historia de AWS

Amazon Web Services (AWS) se lanzó en 2006, comenzando con servicios clave como Amazon S3 y EC2. Desde entonces, AWS ha crecido exponencialmente, ofreciendo cientos de servicios en la nube a millones de clientes globales. Hoy, AWS es la plataforma en la nube más adoptada del mundo, impulsando la innovación en áreas como inteligencia artificial y aprendizaje automático.

TIPOS DE RECURSOS

Computo

AWS proporciona una variedad de servicios de cómputo, como Amazon EC2, que permite a los usuarios ejecutar aplicaciones en instancias virtuales escalables. EC2 proporciona una capacidad de cómputo segura y redimensionable en la nube, lo que facilita la escalabilidad y el manejo de cargas de trabajo variables. Otros servicios de cómputo incluyen AWS Lambda, que permite ejecutar código sin aprovisionar ni gestionar servidores, y Amazon ECS para la gestión de contenedores.

Almacenamiento

AWS ofrece soluciones de almacenamiento escalables y seguras para respaldar datos de cualquier tipo. Amazon S3 es uno de los servicios de almacenamiento más populares, proporcionando almacenamiento de objetos con alta durabilidad y disponibilidad. Amazon EBS ofrece almacenamiento en bloque para usar con instancias EC2. AWS también proporciona soluciones de almacenamiento de archivos como Amazon EFS, que permite compartir almacenamiento entre múltiples instancias EC2.

redes

La infraestructura de redes de AWS incluye servicios como Amazon VPC, que permite a los usuarios crear redes virtuales aisladas dentro de la nube de AWS. Otros servicios importantes son AWS Direct Connect, que establece una conexión de red dedicada entre las instalaciones del cliente y AWS, y Amazon Route 53, un servicio DNS escalable y de alta disponibilidad. Estos servicios permiten una integración segura y eficiente de las redes locales con la nube de AWS.

Software

AWS proporciona una amplia gama de servicios de software para apoyar el desarrollo, la implementación y la gestión de aplicaciones. Amazon RDS ofrece bases de datos relacionales gestionadas. AWS Elastic Beanstalk facilita la implementación y escalado de aplicaciones web y servicios desarrollados con Java, .NET, PHP, Node.js, Python, Ruby, Go y Docker. Además, AWS proporciona herramientas de desarrollo como AWS CodeCommit, CodeBuild, CodeDeploy y CodePipeline para integrar y entregar código de manera continua.
